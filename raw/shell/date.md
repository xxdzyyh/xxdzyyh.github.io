## shell命令 date

很多时候都需要用到日期时间这些，shell中使用 date 命令可以实现

> 获取当前时间

```
$ date                
2019年 9月12日 星期四 15时26分17秒 CST

# 格式化时间 年月日
$ date +"%Y-%m-%d"
2019-09-12

# 格式化时间 年月日时分秒
$ date +"%Y%m%d%H%M%S"
20190912152601
```

> 时间差（mac）

```
# 一天前
$ date -v-1d +"%Y%m%d"
20190911

# 一天后
$ date -v+1d +"%Y%m%d"
20190913
```
