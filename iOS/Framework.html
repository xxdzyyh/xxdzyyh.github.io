  • Static Library
  • Bundle

  • Relocatable Object File

  • 创建一个 Static Framework

  • build 得到一个 Debug-iphonesimulator/SDK.framework，不生成dSYM文件
  • 将这个framework添加到例外一个project,运行模拟器，可以直接查看framework的源码
  • 将 _CodeSignature 文件夹删掉，还是可以看到framework源码
  • 将 framework 项目移动一下位置，看不到源码了，只能看到汇编代码，这说明源码的路径被记录在某个位置

  • 使用Mach-O查看一下App的SDK.framework，可以看到 Section(_DWARF,...)记录了SDK.framework相关的信息
